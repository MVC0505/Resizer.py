**Image Resizer Tool**

This project is a batch Image Resizer tool built using Python. It resizes all images inside a selected input folder and saves the processed images into an output folder. It is useful for preparing images for web uploads, compression, or standardizing resolution across a dataset.

- **Objective**

  Resize and convert multiple images in batch using Python and Pillow.

- **Features**

  Resize all images in a folder

  Supports JPG, PNG, and JPEG formats

  Prints original and new image size for each file

  Automatically creates output folder if missing

  Easy to customize the target size

- **Tools & Libraries**

  Python

  Pillow (PIL)

  OS module

- **Installation**

  Install Pillow library:

      pip install pillow

- **How to Use**

  Place your input images inside the input_images folder.

  Open and run the Jupyter Notebook (image_resizer.ipynb).

  After execution, resized images will appear inside the output_images folder.

  Terminal/Notebook output will show original and resized dimensions for each image.
